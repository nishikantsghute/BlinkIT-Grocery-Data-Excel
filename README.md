📊 BlinkIT Grocery Data Analysis (Excel Dashboard)

This project presents an interactive Excel dashboard built using BlinkIT grocery sales data to analyze business performance and customer trends. The analysis covers key metrics such as total sales, item categories, outlet types, outlet size, location-based performance, and customer ratings.

Key Highlights
Analyzed 8,500+ grocery records
Created interactive dashboards using Excel charts, slicers, and pivot tables
Evaluated sales performance across outlet types and locations
Identified trends in item categories, ratings, and visibility
Generated insights to support data-driven business decisions
Tools Used
Microsoft Excel
Pivot Tables
Charts & Visualizations
Dashboard Design
Data Cleaning & Analysis
Objective

To transform raw grocery data into meaningful visual insights and build a user-friendly dashboard for business analysis.
